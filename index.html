How To Build A Simple RC-Car

My Background Story

I started building this project a few years ago, but never got around to finish it. A while ago i got my hands on the Kit again. Instantly i got a lot of ideas of what i could do with it, about modifications i could add etc.
The purpose of this build isn't to be useful in any way (though if it turns out to be, i wouldn't mind), rather my goal is to learn along the way especially how to program hardware, and how to design/build/test basic circuits.
